# 🎲 Sorteador Dev Club

## 📋 Sobre o Projeto

O **Sorteador Dev Club** é uma ferramenta simples e direta: o usuário informa um número inicial e um número final, clica em **Sortear** e recebe um número aleatório dentro desse intervalo.

Desenvolvido durante as aulas do curso **[Dev Club](https://devclub.com.br)** como projeto prático para treinar lógica com JavaScript e estilização com CSS.

---

## 🚀 Como rodar o projeto

**Opção 1 — Abrir direto:**
```bash
git clone https://github.com/Almirodesouza/Sorteador-Dev-Club.git
-------------------------------------------------------------------------------------------------------------------
cd Sorteador-Dev-Club
```

**Opção 2 — Com Live Server (VS Code):**
1. Instale a extensão **Live Server** no VS Code
2. Clique com o botão direito no `index.html`
3. Selecione **"Open with Live Server"**
